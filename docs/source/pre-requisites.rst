Pre-Requisites Software
===========================

Before you can install **Click2Cloud OpenShift 3 Enterprise Tool for
Visual Studio,** you must download and install the following
pre-requisites software:

1. **Microsoft Visual Studio 2015 Update 1**

   Install `Microsoft Visual Studio 2015 Update
   1 <https://www.visualstudio.com/downloads/download-visual-studio-vs>`__,
   the following SKUs are supported:

   a. Enterprise 2015

   b. Professional 2015

   c. Community 2015

2. **Microsoft ASP.NET and Web Tools 2015 (RC)**

   Install `Microsoft ASP.NET and Web Tools 2015
   (RC) <https://go.microsoft.com/fwlink/?LinkId=627627>`__.

   Note: Please install the corresponding language pack to make sure Web
   Tools works properly.

3. **Git for Windows**

   To download and install Git for Windows, follow instructions
   specified in `Installing Git for Windows <#_Installing_Git_for>`__ or
   watch video `Git on
   Windows <https://www.youtube.com/watch?v=ScSwPf7sBQo>`__

4. **OpenShift Command Line Interface (CLI) – Latest Version**

   To download and install OpenShift 3 CLI, follow instructions
   specified in `Installing OpenShift 3
   CLI <#_Installing_OpenShift_3>`__.

5. **Docker Command Line Interface (CLI) – Latest Version** (Min
   V1.10.0)

   To download and install Docker CLI, follow instructions specified in
   `Installing Docker CLI <#_Installing_Docker_CLI>`__.

.. note::

   You must have sufficient privileges to install software on your computer. Depending on specific user permissions, disabling the User Account Control (UAC) on Windows 7, 8 or 10 operating system may be necessary.
